# Xray SOCKS5 一键安装脚本

一个简单快速的 Xray SOCKS5 代理服务器一键安装脚本。

## 功能特点

- ✅ 一键安装，无需复杂配置
- ✅ 支持用户名密码认证
- ✅ 支持 UDP 协议
- ✅ 自动识别系统架构
- ✅ 系统服务管理
- ✅ 安全可靠（使用官方 Xray 源）

## 系统要求

- CentOS 7+
- Ubuntu 16+
- Debian 8+
- 具有 root 权限
